  -------------------------
  STEP 1 ‚Äî Install Python
  -------------------------

Ensure Python 3.11+ is installed:

    python --version

If not installed, download from: https://python.org

  ---------------------
  STEP 2 ‚Äî Install uv
  ---------------------

If uv is not installed:

    pip install uv

Verify:

    uv --version

  -------------------------------------
  STEP 3 ‚Äî Create Virtual Environment
  -------------------------------------

Inside project root:

    uv venv

Activate:

Windows: .venv

Mac/Linux: source .venv/bin/activate

  -------------------------------
  STEP 4 ‚Äî Install Dependencies
  -------------------------------

    uv pip install fastapi uvicorn langchain langgraph faiss-cpu sentence-transformers groq sqlalchemy python-dotenv pydantic mcp jinja2 pypdf numpy

  -----------------------------------
  STEP 5 ‚Äî Add Environment Variable
  -----------------------------------

Create a .env file in project root:

    GROQ_API_KEY=your_groq_api_key_here

  ----------------------------------------------
  STEP 6 ‚Äî Build FAISS Index (FIRST TIME ONLY)
  ----------------------------------------------

    python build_index.py

This loads PDFs, creates embeddings, and builds FAISS index.

  --------------------------------------
  STEP 7 ‚Äî Run MCP Server (Terminal 1)
  --------------------------------------

    cd mcp_server
    mcp dev server.py

Leave this running.

  -------------------------------------------
  STEP 8 ‚Äî Run FastAPI Backend (Terminal 2)
  -------------------------------------------

From project root:

    uvicorn app.main:app --reload

You should see: http://127.0.0.1:8000

  -----------------------
  STEP 9 ‚Äî Open Chat UI
  -----------------------

Open in browser:

    http://127.0.0.1:8000/

  ---------------
  FULL RUN FLOW
  ---------------

Terminal 1 ‚Üí MCP Server Terminal 2 ‚Üí FastAPI Backend Browser ‚Üí Chat UI

  -----------------
  TROUBLESHOOTING
  -----------------

If FAISS index missing: python build_index.py

If Groq API error: Ensure .env contains GROQ_API_KEY

If MCP JSON error: Use ‚Äòmcp dev server.py‚Äô instead of ‚Äòpython server.py‚Äô
